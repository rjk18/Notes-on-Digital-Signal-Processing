## 先看一个神奇的"错觉"（Aliasing）

理论之前，先看现象。这是理解**采样定理**为何如此重要的第一步。

我写了一段代码，用 **100 Hz 的采样率** 去采集两个不同的信号：
- **图1**：采集一个 **1 Hz** 和一个 **99 Hz** 的正弦波。
- **图2**：采集一个 **1 Hz** 和一个 **101 Hz** 的正弦波。

你猜怎么着？

![Aliasing Demo Image 1](images/alias_1hz_99hz.png)
***图1**：99Hz的信号（蓝色）被100Hz采样后，看起来完全变成了另一个频率的信号（红色圆点）。你能看出它变成了多少Hz吗？*

![Aliasing Demo Image 2](images/alias_1hz_101hz.png)
***图2**：101Hz的信号（蓝色）遭遇了同样的命运。采样点（红色圆点）的连线看起来和某个低频信号一模一样。*

**思考一下**：为什么截然不同的高频信号，采样后却"看起来"一模一样？如果采样系统无法区分它们，会导致什么后果？（这就是"失真"）

这个现象就是**混叠失真**。它直指数字信号处理世界的**第一块基石**：我们必须以超过信号最高频率两倍的速率进行采样，才能完整地保留信息。这就是**奈奎斯特-香农采样定理**。

## 我的学习地图：从困惑到清晰

在熬过了最初的迷茫后，我发现DSP的知识体系可以梳理成一条比较清晰的路径，这也是本仓库的组织脉络：

### 🧱 第一部分：基石 —— 世界如何被数字化？
当我们想让计算机处理声音、图像这些连续信号时，第一步就是**把它们变成数字**。这部分我们解决"如何变"和"变得对不对"的问题。
- **采样与量化**：模拟到数字的桥梁。
- **采样定理（Nyquist Theorem）**：上文演示的"错觉"告诉我们，采样率必须足够高，否则信息会永久丢失（混叠）。**这是数字世界的物理法则。**
- **离散信号与系统**：数字世界里的"信号"长什么样？系统又如何对它们进行操作？

### 🔧 第二部分：核心工具 —— 我们如何分析它？
有了数字信号，我们需要强大的数学工具来"看透"它们。
- **卷积**：描述系统对信号作用的**最核心运算**。理解它，就理解了系统如何处理信号。
- **傅里叶变换家族（DTFT, DFT, FFT）**：这是**最重要的工具**。它让我们能从"随时间变化"的视角，切换到"由哪些频率组成"的视角。频谱分析、滤波设计都离不开它。**FFT是让计算机快速完成这一分析的算法。**
- **Z变换**：分析系统的"放大镜"。它特别适合用来分析和设计**滤波器**（系统的核心之一），能清晰看出系统稳定不稳定，频率响应如何。

### 🎛️ 第三部分：核心应用 —— 如何改造信号？设计滤波器！
知道了信号的频率成分，我们常常想增强需要的、滤除不要的（比如去除噪音）。
- **滤波器设计**：这就是我们的目标。
    - **IIR滤波器**：借鉴了模拟滤波器的设计经验，能用较少的计算实现陡峭的滤波特性。
    - **FIR滤波器**：可以轻松实现**严格的线性相位**（保证信号形状不失真），结构简单稳定。

### 🎯 终极目标：解决实际问题
掌握以上所有，最终是为了：
- **从嘈杂数据中提取有用信息**（滤波、降噪）。
- **识别信号的模式与特征**（频谱分析、检测）。
- **压缩、增强、合成信号**。
- 为通信、音频处理、图像处理、生物医学工程等领域打下基础。
